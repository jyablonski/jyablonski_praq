# Music Theory Overview

## The 12 Chromatic Notes
There are only 12 distinct pitches in Western music:
C, C#/Dflat, D, D#/Eflat, E, F, F#/Gflat, G, G#/Aflat, A, A#/Bflat, B

## Sharps and Flats
- Sharps (#) raise a note by a half-step: C#, D#, F#, G#, A#
- Flats (flat) lower a note by a half-step: Dflat, Eflat, Gflat, Aflat, Bflat
- Enharmonic equivalents: A# = Bflat, F# = Gflat, etc.
- No sharps/flats exist for: B#, E# (would just be C, F) and Cflat, Fflat (would just be B, E)

## Drop B Tuning
- Standard: E-A-D-G-B-E
- Drop B: B-F#-B-E-G#-C#
- First 5 strings tuned down 1.5 semitones, low string down 2.5 semitones
- Creates octave relationship between 6th and 4th strings for easy power chords

## Power Chords
- Two notes: Root + Perfect 5th (7 semitones apart)
- Examples: A5 = A + E, C5 = C + G
- No 3rd interval = neutral sound (neither major nor minor)
- In drop tuning: one finger across multiple strings creates full power chord with octave

## Major vs Minor Chords
### Major Chords
- Root + Major 3rd (4 semitones) + Perfect 5th
- Sound: bright, happy, resolved
- Example: C major = C + E + G

### Minor Chords  
- Root + Minor 3rd (3 semitones) + Perfect 5th
- Sound: darker, sadder, melancholy
- Example: A minor = A + C + E


## Examples

## Power Chords in Drop B

Basic Power Chords (6th + 5th strings):
- B5: Open + Open (B + F#)
- C5: 1st fret + 1st fret (C + G)
- D5: 3rd fret + 3rd fret (D + A)
- F5: 6th fret + 6th fret (F + C)
- G5: 8th fret + 8th fret (G + D)

With Octave (6th + 5th + 4th strings - same frets):
- B5: Open + Open + Open (B + F# + B octave)
- D5: 3rd + 3rd + 3rd (D + A + D octave)

## Major Chords in Drop B

B Major (root position):
- 6th string: Open (B - root)
- 5th string: 6th fret (D# - major 3rd, 4 semitones up from B)
- 4th string: 4th fret (D# - major 3rd octave)
- = B + D# + F#

F Major:
- 6th string: 6th fret (F - root)
- 5th string: 3rd fret (A - major 3rd, 4 semitones up from F)
- 4th string: 8th fret (G#/Aflat - but this gets tricky...)

## Minor Chords in Drop B

B Minor:
- 6th string: Open (B - root)
- 5th string: 4th fret (D - minor 3rd, 3 semitones up from B)
- 4th string: 4th fret (D - minor 3rd octave)
- = B + D + F#
